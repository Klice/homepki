Import root CA

Register a self-signed root CA you minted elsewhere. homepki seals the private key under the in-memory passphrase like any other CA, and you can issue intermediates and leaves under it normally — those new children get homepki's CRL distribution point baked in, so revocation works for everything issued through homepki going forward. Re-uploading the same cert is a no-op (idempotent on SHA-256 fingerprint).
